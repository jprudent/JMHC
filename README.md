JMHC
====

Jérôme's Mahjong Hand Calculator

About
-----

A scoring program for mahjong chinese rules (aka chinese official, MCR, MCO).

Status
------

Don't (yet) blindly trust results :)
But I think I covered all the cases I could think of.
